# rdma_web
REST API for rdma_intern project in php

## Prerequisites
- PHP 8.1.4
- composer
- MySQL

# Getting started 
Clone this project with the following commands:

```
git clone https://github.com/AinaAzrul/rdma_web.git
cd rdma_web
```
Create the configuration and run the program in vscode.
